# Codes for paper Zhou et al. "Hierarchical linguistic predictions and cross-level information updating during narrative comprehension" 

This repository contains the code for the following paper:

Hierarchical linguistic predictions and cross-level information updating during narrative comprehension. Faxin Zhou, Siyuan Zhou, Yuhang Long, Adeen Flinker, Chunming Lu. Communications Biology, in press (2025). Pre-print (https://www.biorxiv.org/content/10.1101/2025.03.27.645665v1).

## Instructions
- Folder "language model" contains the codes for training the representational prediction models;

- Folder "gGLM encoding model" contains the codes for applying the group-based general linear models to the fMRI signals;

- Folder "PC model" contains the codes for predictive coding models, including both the continuous and sparse updating approaches.
